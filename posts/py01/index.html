Bab 1 Struktur Dasar Program

Kategori: Python
Label:

Tanggal: 06-02-2022 20:14:44

Struktur dasar suatu program dibagi menjadi tiga jenis: struktur sekuensial, struktur cabang, dan struktur loop.
Mengenai struktur cabangnya adalah penjelasan program if…elif…else dan operator ternary.
Mengenai struktur loop, kami akan menjelaskan loop traversal for dan loop while tak terbatas.
Namun sebelum itu, izinkan saya menggunakan sedikit ruang untuk memperkenalkan secara singkat jenis data, dll.

1. Tipe data

Tipe data dibagi menjadi tipe numerik, tipe karakter, dan tipe kombinasi.

  • Tipe numerik
    Ada dua jenis angka, bilangan bulat dan desimal, yang disebut bilangan bulat dan bilangan floating point dengan Python.
    Operasi jenis bilangan: +-*/ adalah penjumlahan, pengurangan, perkalian dan pembagian secara berurutan. Ada tiga hal lagi yang perlu diperhatikan: // Dapatkan hasil bagi bilangan bulat, % dapatkan sisanya, ** pangkat
    Ada juga beberapa fungsi bawaan: abs() menemukan nilai absolut, round() putaran, pow() setara dengan **.

  • tipe string
    String adalah barisan terbatas yang diapit tanda petik. Bahasa Mandarin dan Inggris dihitung sebagai satu karakter, itulah yang membuat Python berbeda dari bahasa lain.
    Operasi string dapat dibagi menjadi operasi publik dan operasi unik. Untuk operasi publik, lihat jenis urutan di Bab 3. Operasi uniknya adalah sebagai berikut:
    Fungsi lower() dan upper() masing-masing mewakili semua karakter dalam huruf kecil dan huruf besar. Fungsi replace() dapat menggantikan karakter atau string.

  • Pemformatan tipe string
    print('{1:8>5},{0:,.2f}'.format(1232,'zhk')) # Dalam {x:n}, x adalah nomor seri, dan urutan n adalah enam karakter padding, perataan, panjang maksimum, tanda ribuan, tipe presisi
    print('{:'^30x}'.format(123456) # Tanda kutip tunggal adalah karakter khusus dan perlu di-escape. Tipe heksadesimalnya adalah x

1. Struktur cabang

Struktur sintaks if…elif…else adalah sebagai berikut: , Untuk operator ternary, silakan merujuk ke "Kode Sintaks Penanganan Pengecualian".

Contoh 1: Indeks Massa Tubuh BMI:
BMI, seperti namanya, adalah indeks massa tubuh.
Indikator spesifiknya adalah sebagai berikut:

Kategori Nilai internasional (kg/m²) Nilai domestik (kg/m²)
Tipis <= 18.5 <= 18.5
normal 18.5 ~ 25.0 18.5 ~ 24.0
Gemuk 25.0 ~ 30.0 24.0 ~ 28.0
kegemukan >= 30.0 >= 28.0
BMI dihitung sebagai berikut: BMI = berat badan (kg) / tinggi badan² (m²)

2. Struktur lingkaran

1) Melintasi loop untuk

for i in range(5) : 遍历5次
Anda dapat menggunakan fungsi range(N) untuk melintasi N kali, atau melintasi setiap karakter dari tipe urutan, dan setiap baris file
Dalam for…else…, else dapat dieksekusi setelah loop berakhir secara normal

Contoh 2: Perhitungan berat. Besarnya benda di bulan 16,5% kali besarnya bumi. Berat badan seseorang saat ini adalah 50kg. Manusia bertambah 0,5 kg setiap tahunnya di bumi. Hitunglah berat badan seseorang di bumi dan bulan dalam sepuluh tahun.

Klik pada penghitungan & kode bobot eksplisit/implisit untuk melintasi N kali

Contoh 3: Hitung jumlah kemunculannya. Masukkan serangkaian karakter dan hitung jumlah karakter huruf besar dan kecil, spasi numerik, dan karakter lainnya.

Ini digunakan ketika jumlah loop tidak dapat ditentukan. Itu juga dapat digunakan bersama dengan yang lain. Itu juga dijalankan ketika loop berakhir secara normal.

Contoh 4: Perhitungan pembagi persekutuan terbesar dan kelipatan persekutuan terkecil. Persyaratan: Dua bilangan bulat.
