Bab 11 Ikhtisar Perpustakaan Pihak Ketiga Python
11.1 Arah perayap web
Perayap web adalah program yang mengotomatiskan akses HTTP dan menangkap halaman HTML. Bahasa Python menyediakan beberapa perpustakaan pihak ketiga dengan fungsi perayap web.
11.1.1requests
Pustaka permintaan adalah pustaka pihak ketiga yang ringkas dan sederhana untuk memproses permintaan HTTP. Keuntungan terbesarnya adalah proses penulisan program lebih mirip dengan proses akses URL normal. Perpustakaan ini didasarkan pada perpustakaan urllib3 dari bahasa Python. Metode merangkum fungsi di atas pustaka fungsi lain untuk menyediakan fungsi yang lebih ramah ini sangat umum dalam bahasa Python. Dalam ekosistem Python, siapa pun memiliki kesempatan untuk mengekspresikan pendapatnya dan menunjukkan bakatnya melalui inovasi teknologi atau penciptaan pengalaman.
Pustaka Permintaan mendukung serangkaian fungsi akses tautan yang sangat kaya, termasuk akuisisi nama domain dan URL internasional, koneksi panjang HTTP dan cache koneksi, sesi HTTP dan penyimpanan cookie, verifikasi SSL gaya browser, autentikasi intisari dasar, perekaman cookie pasangan nilai kunci yang valid, dekompresi otomatis, penguraian kode konten otomatis, pengunggahan file dalam potongan, fungsi proxy HTTP(S), pemrosesan batas waktu koneksi, pengunduhan data streaming, dll.
Lebih lanjut tentang perpustakaan permintaan dapat diaksesSintaks instalasi http://www.python-requests.org adalah:

python35 -m permintaan pemasangan pip -i http://pypi.douban.com/simple –terpercaya-hostpypi.douban.com
Pustaka permintaan menyediakan 7 fungsi untuk mewakili permintaan HTTP untuk halaman web. Fungsi umum perpustakaan permintaan:
request.request(): Buat permintaan dan dukung metode dasar dari metode berikut
request.get(): Metode utama untuk mendapatkan halaman web HTML, sesuai dengan GET HTTP
request.head(): Metode utama untuk mendapatkan header halaman web HTML, sesuai dengan HTTP HEAD
request.post(): metode untuk mengirimkan permintaan POST ke halaman web HTML, sesuai dengan HTTP POST
request.put(): metode untuk mengirimkan permintaan PUT ke halaman web HTML, sesuai dengan PUT HTTP
request.patch(): Mengirimkan permintaan modifikasi sebagian ke halaman web HTML, sesuai dengan PATCH HTTP
request.delete(): Kirim permintaan penghapusan ke halaman HTML, sesuai dengan DELETE HTTP

11.1.2scrapy
Scrapy adalah kerangka pengambilan web tingkat tinggi yang cepat yang dikembangkan dengan Python. Berbeda dari fungsi perayap web sederhana, kerangka scrapy itu sendiri berisi beberapa fungsi umum yang harus dimiliki oleh sistem perayap web yang matang. Ini adalah produk setengah jadi. Siapa pun dapat dengan mudah menggunakan fungsi kerangka kerja yang ada sesuai dengan kebutuhannya untuk menerapkan sistem perayap web profesional melalui perluasan sederhana.
Scrapy memiliki beragam kegunaan dan dapat digunakan dalam pembangunan sistem perayap profesional, penambangan data, pemantauan jaringan, dan pengujian otomatis. Scrapy menyediakan antrian URL, akses multi-threaded asinkron, akses terjadwal, integrasi database dan banyak fungsi lainnya. Sistem perayap yang dibangun berdasarkan ini dapat berjalan 7*24 dan memiliki kemampuan pengoperasian tingkat produk.
Untuk informasi lebih lanjut tentang perpustakaan scrapy, silakan kunjungihttps://scrapy.org/

11.2 Arah analisis data
Analisis data merupakan keunggulan PYthon, dengan sejumlah besar perpustakaan pihak ketiga berkualitas tinggi
11.2.1numpy
Numpy adalah perpustakaan pihak ketiga ekstensi perhitungan numerik open source untuk Python, digunakan untuk memproses array multi-dimensi (ndarray) dari tipe data yang sama, disebut sebagai "array". Pustaka ini dapat digunakan untuk menyimpan dan memproses matriks berukuran besar, yang jauh lebih efisien daripada struktur daftar yang disediakan oleh bahasa Python. Numpy menyediakan banyak alat pemrograman numerik tingkat lanjut, seperti operasi matriks, operasi vektor, transformasi data N-dimensi, dll.
Numpy ditulis secara internal dalam bahasa C dan dikemas secara eksternal dalam bahasa Python. Oleh karena itu, saat melakukan operasi data, program Python berbasis numpy dapat mencapai kecepatan pemrosesan yang mendekati kecepatan bahasa C. Numpy juga telah menjadi perpustakaan ketergantungan dasar untuk perpustakaan lain dalam arah analisis data Python, dan telah menjadi "perpustakaan standar" de facto untuk komputasi ilmiah.
Informasi lebih lanjut tentang perpustakaan numpy dapat ditemukan di: http://www.numpy.org/
Sintaks instalasi:

11.2.2scipy
Scipy adalah toolkit Python yang nyaman dan mudah digunakan yang dirancang untuk sains dan teknik. Berdasarkan perpustakaan numpy, ia menambahkan banyak fungsi perpustakaan yang biasa digunakan dalam perhitungan matematika, sains, dan teknik. Ini mencakup banyak modul seperti statistik, optimasi, integrasi, aljabar linier, transformasi Fourier, analisis sinyal, pemrosesan gambar, dan penyelesaian persamaan diferensial biasa. Seperti namanya, scipy terutama digunakan untuk perhitungan ilmiah dan teknik.
Untuk informasi lebih lanjut tentang perpustakaan scipy, silakan kunjungi: https://scipy.org

11.2.3pandas
Pandas adalah perpustakaan pihak ketiga yang penting berdasarkan ekstensi numpy, yang dibuat untuk menyelesaikan tugas analisis data. Pandas menyediakan sejumlah model data standar dan sejumlah besar fungsi serta metode untuk pemrosesan data yang cepat dan nyaman, menyediakan alat yang diperlukan untuk mengoperasikan kumpulan data besar secara efisien.
Pandas pada awalnya dikembangkan sebagai alat analisis data keuangan, oleh karena itu, pandas memberikan dukungan yang baik untuk analisis deret waktu. Nama Pandas berasal dari data panel dan analisis data Python. Data panel adalah istilah untuk kumpulan data multidimensi dalam perekonomian. Pandas menyediakan dua tipe data dasar: Seri dan DataFrame, yang masing-masing mewakili tipe array satu dimensi dan dua dimensi.

11.3 Arah pemrosesan teks
Bahasa Python sangat cocok untuk memproses teks, sehingga sejumlah besar perpustakaan pihak ketiga yang berharga telah dibentuk ke arah ini. Yang umum digunakan antara lain pdfminer, penpyxl, python-docx, dan beautifulsoup4.
11.3.1pdfminer
pdfminer adalah perpustakaan pihak ketiga yang dapat mengekstrak berbagai jenis informasi dari dokumen PDF. Tidak seperti alat terkait PDF lainnya, alat ini dapat memperoleh dan menganalisis data teks PDF sepenuhnya. pdfminer dapat memperoleh posisi pasti, font, jumlah baris, dan informasi lain dari teks dalam PDF, dan dapat mengubah teks PDF menjadi HTML dan format teks. pdfminer berisi dua alat penting: pdf2txt.py dan dumppdf.py.
pdf2txt.py dapat mengekstrak semua konten teks dari file PDF. dumppd.py dapat mengonversi konten file PDF ke format XML dan selanjutnya mengekstrak gambarnya.

11.5 Orientasi Antarmuka Grafis Pengguna
Pustaka standar Python memiliki pustaka GUI bawaan - Tkinter. Perpustakaan ini dikembangkan berdasarkan Tcl/Tk. Namun, perpustakaan ini sudah sangat tua dan memberikan kontrol pengembangan yang sangat terbatas. Gaya GUI yang ditulis jauh dari gaya GUI program modern. Dari perspektif pengalaman pengguna, perpustakaan Tkinter belum matang.
Tiga perpustakaan ekologi Python antarmuka grafis pengguna berkualitas tinggi yang umum digunakan: PyQt5, wxPython, PyGTK
11.5.1PyQt5
PyQt5 adalah pustaka Python pihak ketiga untuk kerangka aplikasi Q5. Ia memiliki lebih dari 620 kelas dan hampir 6.000 fungsi dan metode. Ini adalah perpustakaan pihak ketiga GUI tingkat komersial paling matang dengan Python. Perpustakaan ini saat ini merupakan perpustakaan GUI pihak ketiga terbaik untuk bahasa Python. Ini dapat digunakan lintas platform pada sistem operasi seperti Window, Linux, dan MacOSx.
PyQt5 menggunakan mekanisme "slot sinyal" untuk mengikat peristiwa ke penangan yang sesuai. Sederhananya, sinyal adalah peristiwa dan slot adalah pengendali peristiwa. Misalnya, operasi klik mouse adalah sinyal, dan program yang menangani klik mouse adalah slot. Sebuah sinyal dapat dihubungkan ke beberapa slot atau tidak. Sinyal dapat menghubungkan informasi lain. Formulir PyQt5 memiliki banyak informasi bawaan, dan Anda juga dapat menyesuaikan sinyal.

11.6 Arah pembelajaran mesin
Pembelajaran mesin adalah cabang penting dari bidang kecerdasan buatan, dan bahasa Python juga merupakan bahasa dasar yang penting untuk pembelajaran mesin dan kecerdasan buatan. Tiga framework pembelajaran mesin berkualitas tinggi yang umum digunakan: scikit-learn, TensofFlow, dan Theano.
11.6.1scikit-learn
scikit-learn adalah alat penambangan data dan analisis data yang sederhana dan efisien yang dibangun di atas NumPy, SciP, dan matplotplib. Proyek scikit-learn pertama kali diselenggarakan dan dikembangkan pada tahun 2007 oleh ilmuwan data David Cournapeau. Ini adalah kerangka kerja sumber terbuka yang dikembangkan khusus untuk aplikasi pembelajaran mesin dalam bahasa Python. Fungsi dasar scikit-learn terutama mencakup 6 bagian: klasifikasi, regresi, pengelompokan, reduksi dimensi data, pemilihan model, dan pemrosesan awal data. scikit-learn juga dikenal sebagai sklearn.
Untuk informasi lebih lanjut tentang perpustakaan scikit-learn, silakan kunjungihttp://scikit-learn.org/Sintaks instalasi: python35 -m pipinstall scikit-belajar -i http://pypi.douban.com/simple –terpercaya-hostpypi.douban.com
11.6.2TensorFlow
TensorFlow adalah sistem pembelajaran kecerdasan buatan generasi kedua yang dikembangkan oleh Google berdasarkan DistBelief. Ini juga digunakan untuk mendukung kerangka latar belakang sistem AlphaGo yang terkenal. Namanya berasal dari prinsip operasinya sendiri. Tensor mengacu pada array berdimensi N, dan Flow mengacu pada penghitungan berdasarkan grafik aliran data. TensorFlow menjelaskan proses perhitungan di mana tensor mengalir dari satu ujung grafik aliran ke ujung lainnya.
TensorFlow memiliki beragam aplikasi, mulai dari pengenalan ucapan atau pengenalan gambar hingga terjemahan mesin atau pelacakan otonom. Ini dapat berjalan di pusat data dengan puluhan ribu server, serta di ponsel cerdas atau perangkat yang tertanam.
Informasi lebih lanjut tentang perpustakaan TensorFlow dapat ditemukan dihttp://www.tensorflow.org/
11.6.3Theano
Theano dirancang untuk melakukan operasi pada algoritma jaringan saraf skala besar dalam pembelajaran mendalam dan pandai memproses array multi-dimensi.
Pengembangan Theano dimulai pada tahun 2007. Dapat dipahami bahwa ini adalah kompiler yang mengoperasikan ekspresi matematika dan dapat berjalan secara efisien pada GPU atau CPU. Theano adalah perpustakaan yang condong ke arah pengembangan tingkat rendah, lebih seperti platform penelitian daripada perpustakaan pembelajaran mendalam murni.
Untuk informasi lebih lanjut tentang perpustakaan Theano, silakan kunjungihttp://deeplearning.net/software/theano/Sintaks instalasi: python35-m pip instal theano-i http://pypi.douban.com/simple –host-terpercaya pypi.douban.com
11.7Arah pengembangan web
Pengembangan web adalah arah penting dalam popularitas bahasa Python, yang terutama digunakan untuk pengembangan back-end server. Program kompleks berdasarkan kerangka pengembangan Web Python. Ekosistem Python pihak ketiga yang umum digunakan meliputi: Django, Pyramid, dan Flask.
11.7.1Django
Django adalah kerangka aplikasi web sumber terbuka paling populer di ekosistem Python. Django mengadopsi mode penulisan Model, Templat dan Tampilan, yang disebut mode MTⅣ. Diantaranya model adalah lapisan akses data, yang digunakan untuk menangani segala hal yang berkaitan dengan data, termasuk cara mengakses, cara memverifikasi validitas, perilaku apa saja yang disertakan, dan hubungan antar data, dll; templat adalah lapisan presentasi, yang digunakan untuk menangani fungsi terkait kinerja dan menentukan gaya halaman melalui templat; tampilannya adalah lapisan logika bisnis, yang digunakan untuk mengakses model dan memanggil logika yang relevan dari templat yang sesuai, dan merupakan jembatan antara model dan templat.
Django menyediakan modul yang sering digunakan dalam pengembangan situs web. Konsep pengembangan Django adalah KERING (Jangan Ulangi Diri Sendiri), yang digunakan untuk mendorong pengembangan yang cepat, sehingga mengurangi waktu dan energi yang dihabiskan oleh pemrogram untuk membangun aplikasi web berkinerja tinggi, membentuk solusi satu atap.
Informasi lebih lanjut mengenai perpustakaan Django dapat ditemukan dihttps://www.djangoproject.com/Sintaks instalasi: python35-m pip instal Django-i http://pypi.douban.com/simple –host-terpercaya pypi.douban.com
11.7.2Pyramid
Pyramid adalah kerangka pengembangan aplikasi web Python open source yang umum. Tujuan utamanya adalah untuk memudahkan pengembang Python dalam membuat aplikasi web. Dibandingkan dengan Django, Pyramid adalah kerangka Web Python sumber terbuka yang relatif kecil, cepat dan fleksibel.
Seperti Django, Pyramid masih berorientasi pada aplikasi web skala besar, namun lebih fokus pada fleksibilitas. Pengembang dapat secara fleksibel memilih database, gaya template, struktur URL, dll. yang digunakan.
Informasi lebih lanjut tentang perpustakaan Piramida dapat ditemukan dihttps://trypyramid.com/Sintaks instalasi: python35 -m pip instal piramida -i http://pypi.douban.com/simple –terpercaya-hostpypi.douban.com
11.7.3Flask
Flask adalah kerangka aplikasi web yang ringan. Dibandingkan dengan Django dan Piramida, ini juga disebut kerangka mikro. Sangat mudah untuk mengembangkan aplikasi web menggunakan Flask, dan Anda bahkan dapat membangun situs web kecil hanya dengan beberapa baris kode. Inti dari Flask sangat sederhana dan tidak secara langsung menyertakan lapisan akses abstrak seperti akses database, tetapi didukung melalui modul ekstensi.
Untuk informasi lebih lanjut tentang perpustakaan Flask, silakan kunjungihttp://flask.pocoo.org/Sintaks instalasi: python35-m pip instal labu -i http://pypi.douban.com/simple –host-terpercaya pypi.douban.com
11.8 Arah pengembangan game
Pengembangan game adalah arah yang menarik. Python telah menjadi bahasa pendukung penting di tingkat logika permainan dan implementasi fungsi. Ada tiga perpustakaan ekologi yang umum digunakan dengan Python: Pygame, Panda3D, dan cocos2d.
11.8.1Pygame
Pygame adalah pustaka Python pihak ketiga yang dikemas berdasarkan pustaka SDL dan berorientasi pada pengembangan game tingkat pemula. Selain untuk membuat game juga digunakan untuk membuat aplikasi multimedia. Diantaranya, sDL (Simple DirectMedia Layer) adalah pustaka pengembangan multimedia lintas platform open source yang menyediakan akses mudah ke perangkat keras audio, keyboard, mouse, dan grafis melalui fungsi dasar OpenGL dan Direct3D.
Pygame adalah kerangka pengembangan game yang menyediakan sejumlah besar logika dasar dan dukungan fungsional terkait game. Sangat cocok sebagai perpustakaan pengantar untuk memahami dan mempraktikkan pengembangan game.
Untuk informasi lebih lanjut tentang perpustakaan Pygame, silakan kunjungihttp://www.pygame.org sintaks instalasi: python35 -m pip instal pygame -i http://pypi.douban.com/simple —trusted-hostpypi.douban.com
11.8.2Panda3D
Panda3D adalah perpustakaan rendering 3D dan pengembangan game open source lintas platform. Sederhananya, ini adalah mesin permainan 3D yang dikembangkan bersama oleh Disney dan Pusat Teknologi Hiburan Universitas Carnegie Mellon. Panda3D mendukung Python dan C++, tetapi mendukung Python secara lebih komprehensif.
Panda3D mendukung banyak fitur yang didukung oleh mesin permainan canggih kontemporer, seperti peta normal, peta gloss, HDR, rendering kartun, dan rendering wireframe.
Informasi lebih lanjut tentang perpustakaan Panda3D dapat ditemukan dihttp://www.panda3d.org sintaks instalasi: python35 -m pipinstall panda3d -i http://pypi.douban.com/simple –terpercaya-hostpypi.douban.com
python35 -m pipinstall panda3d -i http://pypi.douban.com/simple –terpercaya-hostpypi.douban.com
11.8.3cocos2d
Cocos2d adalah kerangka kerja untuk membangun game 2D dan aplikasi antarmuka grafis interaktif. Ini mencakup beberapa versi seperti C++, JavaScript, Swift, dan Python. Cocos2d didasarkan pada OpenGL untuk rendering grafis dan dapat menggunakan GPU untuk akselerasi. Mesin Cocos2d menggunakan struktur pohon untuk mengelola objek permainan. Sebuah permainan dibagi menjadi beberapa adegan yang berbeda, dan sebuah adegan dibagi menjadi beberapa lapisan yang berbeda. Setiap lapisan memproses dan merespons peristiwa pengguna.
11.9 Lebih banyak perpustakaan pihak ketiga
Bahasa Python memiliki ratusan ribu perpustakaan pihak ketiga, yang mencakup hampir semua bidang teknologi informasi. Bahkan di setiap arah, akan ada banyak profesional yang mengembangkan beberapa perpustakaan pihak ketiga untuk memberikan desain tertentu. Selain petunjuk yang disebutkan dalam bab ini, perpustakaan pihak ketiga berikut juga biasa digunakan.

11.9.1PIL
Pustaka PIL adalah pustaka pihak ketiga yang penting untuk pemrosesan gambar dalam bahasa Python. Ini mendukung penyimpanan, pemrosesan, dan tampilan gambar. Ini dapat menangani hampir semua format gambar dan dapat menyelesaikan operasi seperti penskalaan, pemotongan, overlay, dan penambahan garis, gambar, dan teks ke gambar. Gunakan bahasa Python untuk memproses program terkait gambar, dan perpustakaan PIL lebih disukai.
Pustaka PIL terutama dapat memenuhi persyaratan fungsional pengarsipan gambar dan pemrosesan gambar.
Pengarsipan gambar: Pemrosesan gambar secara batch, menghasilkan pratinjau gambar, konversi format gambar, dll. Pemrosesan gambar: pemrosesan gambar dasar, pemrosesan piksel, pemrosesan warna, dll.
Informasi lebih lanjut tentang perpustakaan PIL dapat ditemukan dihttp://pillow.readthedocs.io sintaks instalasi: python35 -m pip pasang bantal -i http://pypi.douban.com/simple –terpercaya-hostpypi.douban.com
11.9.2SymPy
SymPy adalah perpustakaan pihak ketiga Python yang mendukung perhitungan simbolik. Ini adalah sistem aljabar komputer berfitur lengkap. Kode SymPy ringkas dan mudah dipahami, serta mendukung perhitungan simbolik, perhitungan presisi tinggi, pencocokan pola, menggambar, penyelesaian persamaan, kalkulus, matematika kombinatorial, matematika diskrit, geometri, pengenalan dan statistika, serta perhitungan dan aplikasi di bidang fisika.

Untuk informasi lebih lanjut tentang perpustakaan SymPy, silakan kunjungi:http://www.sympy.org/

11.9.3NLTK
NLTK adalah perpustakaan pihak ketiga yang sangat penting untuk pemrosesan bahasa alami Python. Ini mendukung banyak bahasa, terutama bahasa Cina. NLTK dapat digunakan untuk pemrosesan korpus, statistik teks, pemahaman konten, analisis sentimen, dan aplikasi lainnya, serta memiliki nilai penerapan yang sangat andal.
Informasi lebih lanjut tentang NLTK dapat ditemukan dihttp://www.nltk.org/
11.9.4WeRoBot
WeRoBot adalah kerangka pengembangan akun publik WeChat, juga dikenal sebagai kerangka robot WeChat. WeRoBot dapat mengurai pesan yang dikirim oleh server WeChat dan mengonversi pesan tersebut menjadi jenis Pesan atau Acara. Diantaranya, mewakili pesan yang dikirim oleh pengguna, seperti pesan teks dan pesan gambar; Peristiwa mewakili peristiwa yang dipicu oleh pengguna, seperti peristiwa perhatian dan pemindaian peristiwa konfirmasi 2D. Setelah penguraian dan konversi pesan selesai. WeRoBot akan mentransfer pesan ke Handler untuk diproses, dan mengirimkan nilai pengembalian Handler ke server WeChat, sehingga mewujudkan fungsi robot WeChat yang lengkap.
