8.1 Ekologi Komputasi
Pergerakan open source dalam 20 tahun terakhir telah menghasilkan sejumlah besar sumber daya yang dapat digunakan kembali yang berakar kuat di berbagai bidang teknologi informasi, yang secara langsung dan efektif mendukung kecepatan perkembangan teknologi informasi di luar bidang teknis lainnya, sehingga membentuk ekosistem komputasi. Penggunaan sumber daya yang dapat digunakan kembali secara ekstensif untuk membangun aplikasi dengan cepat telah menjadi metode pengembangan produk utama di industri. Bahasa Python telah berkomitmen terhadap open source sejak awal, dan telah membentuk ekosistem pemrograman dan komputasi terbesar di dunia.
Situs web resmi Python menyediakan fungsi indeks perpustakaan teman ketiga:http://pypi.python.org/pypi
Halaman ini mencantumkan informasi dasar tentang ratusan ribu perpustakaan pihak ketiga dalam bahasa Python. Pustaka fungsi ini mencakup semua arahan teknis di bidang informasi. Yang perlu diperhatikan di sini adalah tidak semua pustaka fungsi dalam bahasa Python ditulis dengan bahasa Python. Karena Python memiliki metode pemrograman yang sangat sederhana dan fleksibel, banyak perpustakaan profesional yang ditulis dalam c, C+t, dan bahasa lain dapat dienkapsulasi melalui antarmuka sederhana untuk dipanggil oleh program bahasa Python. Fungsi lengket seperti itu menjadikan bahasa Python sebagai antarmuka antara berbagai bahasa pemrograman, dan bahasa Python juga dikenal sebagai "bahasa lem".
Justru karena bahasa Python memiliki daya rekat yang kuat, komunitas bahasa pemrograman terbuka terbesar di dunia dengan cepat terbentuk di sekitarnya, ratusan ribu perpustakaan pihak ketiga dalam skala besar telah didirikan, dan ekosistem komputasi telah dibangun.
Tiga puluh tahun yang lalu, bidang komputer masih berada di era pertanian tebang-bakar, dan program hanya dapat memanggil fungsi API yang disediakan secara resmi. Dua puluh tahun yang lalu, dengan kebangkitan dan perkembangan pesat gerakan open source, sejumlah proyek open source lahir, yang mengurangi kesulitan bagi para profesional untuk menulis program dan memungkinkan penggunaan kembali kode di tingkat profesional. 10 tahun yang lalu, gerakan open source berkembang secara mendalam, dan para profesional mulai menyumbangkan sejumlah besar hasil penelitian dan pengembangan terbaik di berbagai bidang dan mempublikasikannya melalui perpustakaan open source. Bagaimana dengan hari ini? Ekosistem komputasi yang sangat besar telah terbentuk di bidang pemrograman, yang memerlukan bahasa atau metode pemrograman yang dapat menyatukan kode-kode dari bahasa yang berbeda, karakteristik yang berbeda, dan metode penggunaan yang berbeda. Sejarah telah memilih bahasa Python, dan bahasa Python juga telah membuktikan nilainya.
Program pihak ketiga Python menyertakan berbagai nama (perpustakaan), modul, dan paket. Ada sedikit perbedaan antara konsep-konsep ini.
Dalam tutorial ini kita secara kolektif akan menyebut kode-kode yang dapat digunakan kembali ini sebagai "perpustakaan".
8.1.1 Pustaka standar Python
Ada bagian dari ekosistem komputasi Python yang dirilis dengan paket instalasi Python dan dapat digunakan oleh pengguna kapan saja. Ini disebut perpustakaan standar Pytrion. Dibatasi oleh ukuran paket instalasi Python, jumlah pustaka standar tidak terlalu banyak, sekitar 270. Pustaka standar Python umumnya dipasang di direktori Lib.
C:\Users\Administrator\AppData\Local\Programs\Python\Python35
Sebagian dari pustaka standar Python dijalankan dan dikelola oleh PSF (Python Software Foundation), namun sebagian besar pustaka standar Python masih dikelola oleh pengembang aslinya.
8.1.2Perpustakaan pihak ketiga Python
Ekosistem komputasi Python yang lebih luas menggunakan metode instalasi tambahan untuk melayani pengguna, yang dikenal sebagai pihak ketiga Python perpustakaan.好者开发,没有顶层设计,由开发者采用“尽力而为”的方式维护。 Python mengelola instalasi sebagian besar perpustakaan pihak ketiga Python melalui alat instalasi generasi baru pip.
