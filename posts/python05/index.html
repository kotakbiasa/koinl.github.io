Bab 5 Operasi File

Operasi file dasar

buka terbuka

f = open('name', 'mode')
f adalah objek file
nama: string atau jalur nama file target
mode: mode akses: read-only: r, jika file tidak ada, kesalahan akan dilaporkan; dapat dihilangkan
　　　　　 　 　Tulis: w, jika file tidak ada, buatlah;
　　　　　 　 　Tambahkan: a, jika file tidak ada, buatlah;
Jika berisi +, dapat dibaca dan ditulis; jika berisi b, maka dalam format biner.

membaca dan menulis membaca, menulis

membaca:

  • baca(): f.baca(angka)
    Tabel num membaca beberapa byte, jika tidak ditentukan, semua byte
  • garis baca(): f.garis baca()
    Kembali sebagai daftar, setiap baris berisi satu elemen
  • readline(): Membaca konten baris ke-n untuk ke-9 kalinya
    f.tulis('aaa')
  • penunjuk file
  • Apa itu penunjuk file?
  • seek(): f.seek(offset, posisi awal)
    Posisi awal: 0 dimulai dengan 1 dan diakhiri dengan 2

    dekat dekat

    f.tutup()

    Cadangan berkas

    Operasi file dan folder

    Untuk melakukan operasi ini, Anda harus menggunakan fungsi yang relevan dari modul os;
    Jadi, Anda harus mengimpor: import os;
    然后再使用模块相关功能: os.func()
  • dokumen
  • Ganti nama: os.rename(old_name,new_name)
  • menghapus: os.remove(name)
  • map
  • Buat: os.mkdir (nama folder)
  • Hapus: os.rmdir (nama folder)
  • Dapatkan jalur direktori file saat ini: os.getcwd()
  • Beralih direktori: os.chdir (nama direktori baru)
  • Dapatkan daftar direktori: os.listdir()

    Contoh: Ubah nama file dalam batch, tambahkan atau hapus string tertentu
