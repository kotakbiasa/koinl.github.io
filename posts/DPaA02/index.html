Bab 2 Analisis Data (Panda)

Kategori: Data-Processing-and-Analysing
Label:

Tanggal: 07-09-2022 12:05:27

Pustaka Pandas adalah alat berbasis NumPy yang dirancang untuk mencapai tujuan tersebutanalisis dataDibuat untuk tugas itu.

1. Unduh perpustakaan panda

Metode pengunduhan: Masukkan "terminal" di bagian bawah PyCharm:
pip install pandas -i http://mirrors.aliyun.com/pypi/simple/ --trusted-host mirrors.aliyun.com
Selain itu, untuk mengimpor file .xlsx, Anda perlu mengimpor dua paket tambahan: xlrd dan openpyxl.

2. Benda seri

Seri mirip dengan objek array satu dimensi, yang terdiri dari satu kumpulan data dan kumpulan indeks lain yang terkait dengannya.

1. Pembuatan objek Seri

bisa lewatsusunanDankamusuntuk membuat objek Seri.
dapat ditentukan dalam matriks index 。

Klik untuk menampilkan/menyembunyikan kode pembuatan objek Seri

2. Akses ke objek Seri

dapat diakses secara terpisah susunan indeksDan susunan nilai。
Juga, untuk berkunjungTentukan nilai array, bisa dilewatisubskrip dari array utamaUntuk mengaksesnya, Anda juga bisa menggunakan Tag indeks objek SeriAyo berkunjung.
Juga, untuk berkunjungbeberapa elemen, caranya mirip dengan mendapatkan elemen, tetapi sedikit berbeda.

Klik untuk menampilkan/menyembunyikan kode akses objek Seri

3. Operasi penambahan, penghapusan dan modifikasi objek Seri

Operasi penambahan (atau modifikasi) adalah penugasan langsung ke elemen yang ingin Anda tambahkan (atau modifikasi).
Dalam operasi penghapusan,del x['a'] Elemen dengan indeks a dapat langsung dihapus;x.pop('a') Akan mengembalikan konten dari nilai yang dihapus; `x.drop('a')`` mengembalikan array tanpa elemen yang diindeks sebagai a, yaitu mengembalikan array setelah operasi penghapusan,Nilai objek x tidak berubah。

Klik untuk menampilkan/menyembunyikan operasi penambahan, penghapusan dan modifikasi objek Seri

3. Objek DataFrame

Deret mirip dengan objek array dua dimensi

1. Pembuatan objek DataFrame

bisa lewatmatriksDankamusuntuk membuat objek DataFrame
dapat ditentukan dalam matriks index Dan columns, yang dapat ditentukan dalam kamus index(kolom adalah kunci kamus)

Klik untuk menampilkan/menyembunyikan kode untuk membuat objek DataFrame

2. Akses ke objek DataFrame

dapat diakses secara terpisah susunan indeks、susunan kolomDan susunan nilai。
Juga, untuk berkunjungTentukan nilai array, bisa dilewatisubskrip dari array utamaUntuk mengaksesnya, Anda juga bisa menggunakan kolom, tag indeks objek Data FrameAyo berkunjung.
Juga, untuk berkunjungbeberapa elemen, caranya mirip dengan mendapatkan elemen, tetapi sedikit berbeda.

Klik untuk menampilkan/menyembunyikan kode akses objek Seri

3. Menambah, menghapus, dan mengubah operasi pada objek DataFrame

Operasi penambahan (atau modifikasi) adalah penugasan langsung ke elemen yang ingin Anda tambahkan (atau modifikasi).
Dalam operasi penghapusan,del x['a'] Elemen dengan indeks a dapat langsung dihapus;x.pop('a') Akan mengembalikan konten dari nilai yang dihapus; `x.drop('a')`` mengembalikan array tanpa elemen yang diindeks sebagai a, yaitu mengembalikan array setelah operasi penghapusan,Nilai objek x tidak berubah。

Klik untuk menampilkan/menyembunyikan kode operasi penambahan, penghapusan, dan modifikasi objek DataFrame

4. Penggunaan dasar NumPy

1. Statistik

1) Nilai dan penjumlahan maksimum

Jika diperlukanper kolomUntuk nilai maksimum dan minimum, gunakan fungsi idxmax() dan idxmin(),axis=1 参数 dapat diubah menjadidemi barisDapatkan nilai. ;
Jika perlu tekanDaftarUntuk menjumlahkannya, gunakan fungsi sum(), jika perluKumulatif berdasarkan kolomSingkatnya, gunakan fungsi cumsum(),axis=1 参数 dapat diubah menjadidemi barisDapatkan nilai.

Klik untuk menampilkan/menyembunyikan statistik data · Kode untuk nilai maksimum dan penjumlahan

2) Hapus duplikat

Ada dua fungsi utama untuk deduplikasi: unik() dan value_counts().
Fungsi dari fungsi unik() adalahHapus elemen duplikat, jika parameternya adalah objek Seri, array NumPy akan dikembalikan.
Fungsi value_counts() dapat melakukannyaMengembalikan objek Seri,di dalam index ke dalam objek Seri asliElemen yang tidak berulang， values adalah jumlah kemunculan elemen yang tidak berulang.
